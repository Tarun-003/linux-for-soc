# 🌐 Networking Commands (Linux for SOC)

SOC analysts use networking commands to monitor connections, analyze traffic, and detect suspicious activity.

---

## 📌 Why Networking Commands Matter?

- Identify active connections
- Detect suspicious IP addresses
- Monitor open ports
- Investigate attacks

---
