# ⚙️ Process Monitoring (ps, top) – Linux for SOC

Processes are running programs in a system.

SOC analysts monitor processes to detect suspicious or malicious activity.

---

## 📌 What is a Process?

A process is an active program running on a system.

Example:
- Browser
- SSH service
- Background applications

---

## 🛠 1. ps Command (Process Snapshot)

Shows currently running processes.
