# 📂 Linux Log Files for SOC Analysts

Linux systems store important logs in the /var/log directory.

SOC analysts use these logs to detect and investigate security incidents.

---

## 📁 What is /var/log?

It is the directory where system and application logs are stored.

---

## 📊 Important Log Files

| File | Description |
|------|------------|
| /var/log/auth.log | Login & authentication logs |
| /var/log/syslog | System activity logs |
| /var/log/kern.log | Kernel logs |
| /var/log/dmesg | Boot messages |
| /var/log/apache2/access.log | Web server access logs |

---

## 🔍 Viewing Logs

```bash
cat /var/log/auth.log
less /var/log/syslog
tail /var/log/auth.log
