# 🐧 Linux Basics for SOC Analysts

Linux is widely used in cybersecurity for monitoring, analysis, and system management.

SOC analysts use Linux to investigate logs, detect threats, and analyze systems.

---

## 📌 Why Linux is Important for SOC?

- Most servers run on Linux
- Security tools are Linux-based
- Logs are stored and analyzed in Linux
- Used for forensic analysis

---

## 🧰 Common Use Cases in SOC

- Log analysis
- File inspection
- Process monitoring
- Network troubleshooting

---

## 💻 Basic Linux Commands

| Command | Description |
|--------|------------|
| ls | List files |
| cd | Change directory |
| pwd | Show current directory |
| cat | View file content |
| clear | Clear terminal |
| mkdir | Create directory |
| rm | Remove file |

---

## 🔍 File Viewing Commands

| Command | Use |
|--------|-----|
| cat | View full file |
| less | Scroll through file |
| head | First few lines |
| tail | Last few lines |

---

## 🛡 Why These Commands Matter in SOC?

- Read log files
- Investigate incidents
- Analyze suspicious files
- Navigate system quickly

---

## 📌 Example

View login logs:
tail /var/log/auth.log

---

## ✅ Summary

Linux is essential for SOC analysts to monitor systems, analyze logs, and investigate security incidents.
