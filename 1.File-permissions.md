# 🔐 Linux File Permissions (SOC Basics)

File permissions control who can read, write, or execute files in a Linux system.

They are essential for maintaining system security.

---

## 👥 Types of Users

Each file has permissions for:

- Owner (user)
- Group
- Others (everyone else)

---

## 📊 Permission Types

| Symbol | Meaning |
|--------|--------|
| r | Read |
| w | Write |
| x | Execute |

---
